Helpdesk and Support Tickets Plugin built only for WordPress. No subscriptions, responsive, front end content, email notifications included.

== Description ==

This plugin has been redesigned and rewritten almost from scratch. Please do not place negative reviews, but help us to locate the problems so we can fix them, instead. Appreciate your collaboration!

Helpdesk system to allow your Customers to create Support Tickets from your website. 

* Built only for Wordpress websites.
* No subscriptions, no limitations, no registrations: install and go!
* Tickets search, ordering, filtering by Status.
* Create Tickets on Customer's behalf.
* Add Messages and receive Customer's messages.
* Email notifications included.
* Attach files to Tickets.
* Front-end included: let your Customer open Tickets from your website!
* Responsive, works on mobile, fast AJAX structure.
* Install wizard will configure the plugin in 1 minute.
* Exensions coming soon!

NEW! 29/04/2017: Added Colors for different Ticket Statuses

== Installation ==

* Login as Administrator of your portal;
* Navigate to Plugins -> Add New;
* Click on Upload;
* Select and Upload the .zip file you downloaded from this website;
* Activate the new plugin.
* Short code for Front End: [helpdesk-support-tickets]
